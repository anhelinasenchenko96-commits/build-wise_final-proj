# BuildWise Landing Page

BuildWise is a responsive landing page for a construction services company.  
The project was created as a final front-end learning project using HTML, CSS, and JavaScript.

## Features

- Responsive layout for desktop, tablet, and mobile screens
- Semantic HTML structure
- Adaptive images and layout sections
- Header navigation with mobile burger menu
- Portfolio carousel
- Partners carousel
- FAQ accordion
- Contact forms with phone number validation
- Accessible form labels and interactive controls
- Hover, focus, active, disabled, and error states

## Technologies

- HTML5
- CSS3
- JavaScript ES6+
- CSS Grid
- Flexbox
- Google Fonts
- libphonenumber-js

## How to run the project

Open index.html in a browser.

Or use a local development server, for example the Live Server extension in Visual Studio Code.
